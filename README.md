## Install dotfile

#### via curl

```bash
sudo sh -c "$(curl -fsSL https://raw.githubusercontent.com/tankibaj/dotfiles-ubuntu/master/install.sh)"
```

#### via wget

```bash
sudo sh -c "$(wget -O- https://raw.githubusercontent.com/tankibaj/dotfiles-ubuntu/master/install.sh)"
```


## Source .zshrc

```bash
source $HOME/.zshrc
```
